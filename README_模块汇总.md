## ❓ 常见问题

### Q: 为什么要先做供应商管理？
**A:** 供应商管理是应付账款的基础。没有供应商信息，应付账款功能无法正常运行。就像盖房子必须先打地基一样。

### Q: 供应商和产品哪个先做？
**A:** 两者都是基础模块，顺序可以灵活调整。但建议：
1. 先做产品管理（知道卖什么）
2. 再做供应商管理（知道从谁那买）
3. 最后做采购管理（开始采购）

### Q: 账期和信用额度是干什么用的？
**A:** 
- **账期**：约定多少天后付款（如30天后付款）
- **信用额度**：最多可以欠多少钱（如最多欠10万）
- 这两个用于应付账款管理

### Q: 停用的供应商可以恢复吗？
**A:** 可以！编辑供应商 → 打开"合作中/已停用"开关 → 保存即可。
