# 组装电脑报价单 - 智能兼容性检测系统

## 📋 功能概述

这是一个智能的组装电脑报价单系统，主要功能包括：

1. **自动加载默认配置** - 创建报价单时自动带出完整的电脑配件清单（CPU、主板、内存、硬盘等10项）
2. **智能兼容性检测** - 选择CPU后自动过滤兼容的主板和内存，不兼容的配件会用红色背景标注
3. **实时价格计算** - 自动计算小计和总计
4. **可视化反馈** - 兼容的配件用绿色标注，不兼容的用红色标注

## 🚀 实施步骤

### 步骤 1: 执行数据库迁移

运行 `migration.sql` 文件，这将：
- 为 `products` 表添加 `platform_tags` 字段
- 插入示例产品数据（Intel和AMD平台的CPU、主板、内存）
- 创建默认配置模板表

```sql
-- 在 MySQL 中执行
mysql -u your_username -p bjd < migration.sql
```

### 步骤 2: 部署新文件

将以下文件上传到服务器：

1. **quote_create_v2.php** - 改进的报价单创建页面
2. **get_products_v2.php** - 产品数据获取接口
3. **product_edit_v2.php** - 改进的产品编辑页面
4. **product_save_v2.php** - 产品保存接口（含标签）

### 步骤 3: 添加产品兼容性标签

进入产品管理页面，编辑现有的CPU、主板、内存产品，设置兼容性标签：

#### CPU 标签示例：
- **Intel i5-13400F**: `Intel`, `LGA1700`, `DDR5`, `DDR4`
- **AMD Ryzen 7 7800X3D**: `AMD`, `AM5`, `DDR5`

#### 主板标签示例：
- **华硕 B760M**: `Intel`, `LGA1700`, `DDR5`
- **微星 B650M**: `AMD`, `AM5`, `DDR5`

#### 内存标签示例：
- **金士顿 DDR5 32GB**: `DDR5`
- **金士顿 DDR4 16GB**: `DDR4`

### 步骤 4: 测试兼容性检测

1. 访问 `quote_create_v2.php?template=assembled_pc`
2. 页面会自动加载10项默认配件
3. 在第一行选择一个 Intel CPU
4. 观察主板和内存的下拉选项是否自动过滤
5. 尝试选择不兼容的主板，应该会显示红色背景和警告

## 📊 兼容性规则说明

### 标签体系

系统使用标签数组来定义兼容性，存储在 `products.platform_tags` 字段（JSON格式）：

```json
// CPU 标签
["Intel", "LGA1700", "DDR5"]

// 主板标签  
["Intel", "LGA1700", "DDR5"]

// 内存标签
["DDR5"]
```

### 兼容性检测逻辑

1. **CPU → 主板**：主板必须包含CPU的品牌（Intel/AMD）和接口标签
2. **CPU → 内存**：内存必须包含CPU支持的内存类型标签（DDR4/DDR5）
3. **其他配件**：硬盘、显卡、电源等通用配件无需设置标签

### 兼容性判断示例

✅ **兼容**：
- CPU: `["Intel", "LGA1700", "DDR5"]`
- 主板: `["Intel", "LGA1700", "DDR5"]`
- 内存: `["DDR5"]`

❌ **不兼容**：
- CPU: `["Intel", "LGA1700", "DDR5"]`
- 主板: `["AMD", "AM5", "DDR5"]` ← 品牌不匹配
- 内存: `["DDR4"]` ← 内存类型不匹配

## 🎯 使用流程

### 创建报价单

1. 点击"创建报价单"选择"组装电脑"模板
2. 系统自动加载10项默认配件清单
3. 选择客户和基本信息
4. 从第一行开始选择CPU
5. 系统自动过滤并显示兼容的主板和内存选项
6. 继续选择其他配件（硬盘、显卡等）
7. 系统实时计算价格
8. 保存报价单

### 产品管理

1. 添加/编辑产品时，系统会自动检测产品分类
2. 如果是CPU、主板、内存，会显示"兼容性标签"卡片
3. 勾选相应的兼容性标签
4. 保存产品

## ⚠️ 注意事项

1. **只有组装电脑模板**使用兼容性检测，其他模板（品牌整机、弱电工程等）不需要
2. **标签大小写敏感**，请统一使用规范：`Intel`, `AMD`, `LGA1700`, `AM5`, `DDR5`, `DDR4`
3. **多标签产品**：部分CPU和主板同时支持DDR4和DDR5，可以勾选多个标签
4. **通用配件**：硬盘、显卡、电源、机箱等无平台限制的配件不需要设置标签

## 🐛 故障排查

### 问题：兼容性检测不工作

**检查清单：**
1. 产品是否设置了正确的标签？
2. 标签格式是否正确（JSON数组）？
3. 浏览器控制台是否有JavaScript错误？
4. `get_products_v2.php` 是否正确返回标签数据？

### 问题：下拉选项为空

**可能原因：**
1. 数据库中没有对应分类的产品
2. 产品的 `is_active` 字段为0
3. 标签过滤太严格，没有匹配的产品

**解决方法：**
打开浏览器控制台，查看 `allProducts` 对象和兼容性检查日志
